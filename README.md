# Путешествия по России

Эта проектная работа про путешествия по России от Яндекс Практикума я сверстал макет из фигмы и сделал адаптив.

## В проекте используются:

1. HTML
   * Семантические теги
   * Файловая структура БЭМ Nested
2. CSS
   * Flexbox
   * Grid
   * Адаптив
